= Validating the environment
include::_attributes.adoc[]

これでワークベンチに接続できたので、クラスター内で想定されるすべてのサービスが適切に応答していることを確認しましょう。

ワークベンチで:

. 左側のナビゲーション メニューで、`parasol-insurance/lab-materials/02` というフォルダーに移動します。

. `02-05-validating.ipynb` というノートブックを開きます。

. Jupyter Notebook でセルを実行したことがない場合は、次の手順に従います。

.. [**Restart kernel and Run all Cells**] リンクをクリックします:
+
[.bordershadow]
image::02/02-05-restart-and-run.png[]
.. **Restart** をクリックします:
+
[.bordershadow]
image::02/02-05-restart-kernel.png[]
+
. これらのセルを実行すると、ラボに必要なすべてのサービスが応答していることが確認できます。

出力は次のようになります:

[source,console]
----
Success: Minio is reachable on minio.ic-shared-minio.svc.cluster.local:9000
Success: Postgres Database is reachable on claimdb.ic-shared-db.svc.cluster.local:5432
Success: LLM Service is reachable on granite-7b-instruct-predictor.ic-shared-llm.svc.cluster.local:8080
Success: LLM Service-FlanT5 is reachable on llm-flant5.ic-shared-llm.svc.cluster.local:3000
Success: ModelMesh is reachable on modelmesh-serving.ic-shared-img-det.svc.cluster.local:8033
Success: Milvus Vector DB is reachable on vectordb-milvus.ic-shared-milvus.svc.cluster.local:19530
----

このノートブックの出力が疑わしい場合は、{ic}を主導する担当者にお知らせください。
