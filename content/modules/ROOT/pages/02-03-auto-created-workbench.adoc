ワークベンチ タブに移動してください。

事前に作成されたコンポーネントの 1 つがワークベンチです。

* ワークベンチの横にある矢印をクリックすると、ワークベンチに含まれているライブラリの一部と、割り当てられているリソースの量を確認できます。
+
[.bordershadow]
image::02/02-03-pre-created-workbench-info.png[]

* ワークベンチに接続するには、 **Open** リンクをクリックします。:
+
[.bordershadow]
image::02/02-03-open-link.png[]

* そして:

** 以前と同じ認証情報で認証する:
*** アカウント ID: `{user}`
*** パスワード: `{password}`
* 以下の設定に同意するよう求められます:
+
[.bordershadow]
image::02/02-02-accept.png[]

* *Allow selected permissions* をクリックします。
* これで次の画面が表示されます:
+
[.bordershadow]
image::02/02-03-pre-created-jupyter.png[]

* **parasol-insurance** という追加のフォルダーに注意してください。 +
* このフォルダーは github からクローンされたもので、このラボのメインの作業フォルダーになります。
