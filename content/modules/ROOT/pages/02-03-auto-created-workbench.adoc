= Pre-Created workbench
include::_attributes.adoc[]

ワークベンチ タブに移動してください。

事前に作成されたコンポーネントの 1 つがワークベンチです。

* ワークベンチの横にある矢印をクリックすると、ワークベンチに含まれているライブラリの一部と、割り当てられているリソースの量を確認できます。
+
[.bordershadow]
image::02/02-03-pre-created-workbench-info.png[]

* ワークベンチに接続するには、 **Open** リンクをクリックします。:
+
[.bordershadow]
image::02/02-03-open-link.png[]

* そして:

** 以前と同じ認証情報で認証する:
*** アカウント ID: `{user}`
*** パスワード: `{password}`
* 以下の設定に同意するよう求められます:
+
[.bordershadow]
image::02/02-02-accept.png[]

* *Allow selected permissions* をクリックします。
* これで次の画面が表示されます:
+
[.bordershadow]
image::02/02-03-pre-created-jupyter.png[]

* **parasol-insurance** という追加のフォルダーに注意してください。 +
* このフォルダーは事前に github からクローンされたものですが、説明が全て英語です。説明を日本語化したい場合には、以下の手順を実行ください。
** 既存のparasol-insuranceフォルダを右クリックし、表示されたメニューから"Delete"を選択して削除します。
+
[.bordershadow]
image::02/delete-original.png[]
+
** Jupyter で Git UI を開きます:
+
[.bordershadow]
image::02/git-clone-1.png[]
+
** Git リポジトリの URL を入力します:
+
[.console-input]
[source,adoc]
[subs=attributes+]
----
{git-clone-repo-url}
----
+
[.bordershadow]
image::02/git-clone-2.png[]
** 表示されるフォルダーのうち、"parasol-insurance"をダブルクリックして内容を表示します:
+
[.bordershadow]
image::02/02-03-pre-created-jupyter.png[]
** 左のアイコンから再度Gitアイコンをクリックし"Current Branch"の右にある三角マークをクリックします:
+
[.bordershadow]
image::02/git-branch2.png[]
** 表示されたブランチの中から、"translation-jp"ブランチ(origin/translation-jp)をクリックして"Current Branch"が”translation-jp”に切り替わることを確認します:
+
[.bordershadow]
image::02/git-branch3.png[]
** 左のアイコンの最上部のフォルダーアイコンをクリックします:
** 上記手順により、このラボの説明内容が日本語化されました。
