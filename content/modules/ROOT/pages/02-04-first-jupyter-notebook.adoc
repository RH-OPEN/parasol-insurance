= Running code in a notebook
include::_attributes.adoc[]

NOTE: すでに Jupyter Notebook に慣れている場合は、次のセクションに進んでください。

ノートブックは、フォーマットされたテキストやコードを表示できる _cells_ がある環境です。

これは空のセルです:

[.bordershadow]
image::02/02-05-cell.png[Jupyter Cell]

これはコードが書かれたセルです:

[.bordershadow]
image::02/02-05-cell_code.png[Jupyter Cell Code]

コード セルには、対話的に実行できる Python コードが含まれています。コードを変更してから実行できます。コードはコンピューターやブラウザーではなく、接続している環境 (この場合は *{rhoai}*) で直接実行されます。

コード セルは、ノートブック インターフェイスまたはキーボードから実行できます。

* *From the user interface:* セルを選択し (セル内またはセルの左側をクリック)、ツールバーから  *Run*  をクリックします。
+
[.bordershadow]
image::02/02-05-run_button.png[Jupyter Run]

* *From the keyboard:* `CTRL`+`ENTER` を押してセルを実行するか、`SHIFT`+`ENTER` を押してセルを実行し、次のセルを自動的に選択します。

セルを実行すると、次の例に示すように、コードの結果とセルが実行された時刻に関する情報が表示されます。:

[.bordershadow]
image::02/02-05-cell_run.png[Jupyter run cell]

ノートブックを保存すると、コードと結果が保存されます。ノートブックを再度開いて結果を確認することができます。プログラムを再実行する必要はありませんが、コードには引き続きアクセスできます。

ノートブックは物理的な _ノートブック_ のようなものなので、このように名付けられています。実験 (これから行う) に関するメモや、設定したパラメーターを含むコード自体のメモを取ることができます。実験の出力 (これは実行後のセルの結果です) をインラインで表示し、メモしたいすべてのメモも表示できます (これを行うには、メニューからセルの種類を `Code` から `Markdown` に切り替えます)。

== 試してみる

基本がわかったら、ぜひ試してみてください!

=== 手順

ワークベンチで:

. 左側のナビゲーション メニューで、`parasol-insurance/lab-materials/02` というフォルダーに移動します。

. `02-04-first-jupyter-notebook.ipynb` というノートブックを開きます。

. たとえば、既存のセルを実行したり、セルを追加したり、関数を作成したりして実験します。
+
好きなことをすることができます。これはあなたの環境であり、何かを壊したり他のユーザーに影響を与えたりするリスクはありません。この環境の分離も、{rhoai} によってもたらされる大きな利点です。
. オプションで、Python 3 カーネルを使用してコード セルが実行される新しいノートブックを作成します。:
.. *File ->New ->Notebook*  を選択するか、ランチャー ウィンドウのノートブック セクションで Python 3 タイルをクリックして、新しいノートブックを作成します。:
+
[.bordershadow]
image::02/02-05-new_notebook.png[alt text]

ノートブックでは、異なる言語やバージョンの異なるカーネルを実行できます。
